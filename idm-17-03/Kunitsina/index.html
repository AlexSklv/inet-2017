Как это создавалось...

Домашним заданием было зарегистрироваться на GitHub, после чего необходимо было сделать fork проекта stankin/inet-2017 к себе в хранилище и склонировать на локальную машину. На которой проводились все изменения файлов после чего они были запушены в удаленный репозиторий и отправлен pull/request преподавателю на слияние текущих изменений с веткой master исходного.

№ п/п Наименование Описание
1 Заполнение "index.html" для заполнения и создания сетки воспользуемся тегами
2 Заполнение "css/style.css" сделаем краисые элементы и фичи
3 Изучение рыботы с гит сслыка на исходный проект на GitHub
4 Изчуение консоли с помощью простейших команд git pull, psuh, clone будем работать с консолью git
