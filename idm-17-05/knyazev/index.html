DHCP - Dynamic Host Configuration Protocol.

Сетевой протокол, позволяющий устройству получать информацию об IP адресе и других сетевых параметрах. Протокол является "клиент-серверным"

Процесс получения состоит из 4 этапов: DHCDISCOVER-DHCOFFER-DHCREQUEST-DHCPACK; Обнаружение - предложение - запрос - подтверждение
